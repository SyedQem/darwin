`darwin` is a secure, clean, and modern campus-only student marketplace. Built exclusively for verified students, it features student verification, in-app messaging, referral platform and much more. 


Key Features

### 1. Verification & Student Onboarding
- A immersive, multi-step onboarding wizard (`/onboarding`) gathering:
- University selection with campus logos (currently supporting **Carleton University**, **University of Ottawa**, and **Algonquin College**).
- Academic Profile details (Level of study: Undergraduate, Graduate, PhD, Alumni; Program/Degree).
- Selected interests (Textbooks, Electronics, Furniture, etc.) to customize the student's feed.

### 2. Referral Platform
- **Referral System:** Students get a unique referral code. Referring **5 students** automatically unlocks priority access.
- **Founding Access Spot Purchases:** Monetization tiers powered by Stripe

### 3. Student Marketplace & Listings
- **Browse Feed:** Dynamic search, sorting, and category filters (Textbooks, Electronics, Furniture, Clothing, Tickets, Notes & Study Material, Sports & Fitness, Other).
- **Selling Wizard:** Form validation for student postings, category matching, item condition details (New, Like New, Good, Fair), and image upload capability.
- **Item Listing Detail Page:** Rich product details, university location, seller rating, and reviews.

### 4. Direct Messaging & Chat (`/messages`)
- In-app messaging system between student buyers and sellers.
- Dynamic page-level chat threads, real-time message bubbles, unread notification badges.
